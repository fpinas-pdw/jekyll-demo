Azure Application Gateway es un balanceador de carga de aplicaciones web (capa 7) que permite gestionar el tráfico hacia tus aplicaciones web de manera inteligente y segura.

## Características clave

Application Gateway ofrece capacidades avanzadas como:
- Enrutamiento basado en URL
- SSL/TLS offloading
- Web Application Firewall (WAF)
- Autoscaling
- Zonas de disponibilidad

## Web Application Firewall

Una de las características más valiosas es el WAF integrado, que protege tus aplicaciones contra:
- Inyección SQL
- Cross-site scripting (XSS)
- Ataques de OWASP Top 10
- Bots maliciosos

## Casos de uso

Application Gateway brilla en escenarios como:
- Aplicaciones web de producción que requieren seguridad avanzada
- Arquitecturas de microservicios con múltiples backends
- Aplicaciones que necesitan terminación SSL centralizada
- Entornos que requieren distribución de carga inteligente

La combinación de rendimiento y seguridad hace de Application Gateway una opción preferida para aplicaciones empresariales críticas.
